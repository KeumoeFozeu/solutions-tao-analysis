# Solutions to Tao's Analysis I

This repository contains the solutions to _Analysis I_ by Terence Tao, up to Chapter 7.
This work is an independent study project.
It is intended to be a supplemental resource for students working through the textbook.

The solutions are typeset in LaTeX and compiled into a single PDF that follows the structure, style and notation of the textbook.
The remaining chapters and _Analysis II_ may be covered in the future.

### File
The full solution set (Chapters 1 – 7): [`tao_analysis_solutions.pdf`](tao_analysis_solutions.pdf) 


### Referenced Work
Tao, T. (2022). _Analysis I: Fourth Edition_. Texts and Readings in Mathematics, 37. Hindustan Book
